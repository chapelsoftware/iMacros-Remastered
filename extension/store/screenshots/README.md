# Screenshots for Chrome Web Store

This directory should contain screenshots for the Chrome Web Store listing.

## Requirements

Chrome Web Store requires:
- At least 1 screenshot (up to 5 recommended)
- Size: 1280x800 or 640x400 pixels
- Format: PNG or JPEG
- No alpha/transparency

## Recommended Screenshots

Create the following screenshots:

### 1. main-panel.png
- Show the iMacros side panel open
- Display the macro list with a few example macros
- Highlight the Record/Play buttons

### 2. recording.png
- Show the extension actively recording
- Display the recording indicator
- Show a form being filled

### 3. macro-editor.png
- Show the macro editor with code
- Display syntax highlighting
- Show a complete macro example

### 4. playback.png
- Show a macro being executed
- Display the progress indicator
- Show the results

### 5. settings.png
- Show the options/settings page
- Display available configuration options

## Tips

- Use a clean browser profile
- Hide bookmarks bar for cleaner look
- Use example macros that demonstrate key features
- Ensure text is readable at thumbnail size
- Use consistent window size across all screenshots

## Promotional Images (Optional)

- Small tile: 440x280 pixels
- Large tile: 920x680 pixels
